# MajorFinanceira — Constituição do Projeto

## Visão
Landing page de alta conversão para a MajorFinanceira, correspondente bancário de crédito consignado. Objetivo único: transformar visitantes em conversas no WhatsApp.

## Empresa
- **Nome:** MajorFinanceira
- **CNPJ:** 14.626.010/0001-41
- **Tipo:** Correspondente Bancário Autorizado
- **Bancos parceiros:** BMG, Banco Pan, Facta Financeira, Caixa Econômica Federal, Banco do Brasil, Itaú Consignado

## Público-Alvo
1. Aposentados e Pensionistas do INSS
2. Servidores Públicos Civil e Militar
3. Trabalhadores CLT (Setor Privado)
4. Beneficiários do BPC (Benefício de Prestação Continuada)

## Sensação-alvo
**Segurança e Confiança** primeiro → **Acolhimento** logo depois.

## Estrutura de Seções (em ordem)
1. **Hero** — Headline forte + CTA WhatsApp
2. **Botão flutuante** — WhatsApp fixo, sempre visível
3. **Quem pode contratar** — 4 perfis (INSS, CLT, Servidor, BPC)
4. **Como funciona** — 3 passos simples
5. **Vantagens** — Diferenciais reais da MajorFinanceira
6. **Bancos parceiros** — Logos (credibilidade)
7. **Depoimentos** — 3 placeholders para substituir por reais
8. **FAQ** — 7 perguntas mais comuns do consignado
9. **CTA Final** — WhatsApp novamente
10. **Footer** — CNPJ + "Correspondente Bancário Autorizado"

## Stack
- **Frontend:** Next.js 16 + TypeScript + Tailwind CSS
- **Deploy:** Vercel (auto-deploy na branch main)
- **Banco de dados:** Não necessário
- **Repositório:** https://github.com/eusoukleberpereira-cyber/majorfinanceira
- **Vercel:** https://vercel.com/kleber-pereiras-projects

## Objetivo de Conversão
Visitante → Conversa no WhatsApp (sem formulário, sem banco de dados)
